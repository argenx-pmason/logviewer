Once that is done you will be able to use it:

  1. go to LSAF PROD or LSAF TEST
  2. Go to the Repository view (click on Repository icon in the left vertical toolbar)
  3. Either:
    • File
      1. Find a file you want to look at (currently supports these file types: js, job, css, cmd, mnf, html, lst, txt, csv, sas, json, svg, png, jpg, pdf, xlsx)
      2. Select it, by clicking next to it (shows a tick)
      3. Click on the bookmark, which will open a new tab to show the file
    • Folder
      1. Navigate to a folder you want to examine
      2. Click on the bookmark, which will open a new tab to show the contents of the folder
